HORDE DEFENSE SPRITE ASSETS
===========================

Place your AI-generated sprites in the appropriate folders.
Recommended size: 128x128 or 256x256 pixels with transparent background.
Format: PNG with transparency

FOLDER STRUCTURE:
-----------------

/towers/
  - grunt.png      (Orc warrior with club)
  - archer.png     (Orc archer with bow)
  - berserker.png  (Large orc with dual axes)
  - shaman.png     (Orc shaman with staff)
  - warlord.png    (Armored orc with banner)
  - siege.png      (Orc with ballista)

/enemies/
  - squire.png         (Young knight trainee)
  - knight.png         (Armored knight)
  - archer.png         (Human archer)
  - cavalry.png        (Knight on horseback)
  - mage.png           (Human wizard)
  - knight_commander.png (Boss - Wave 5)
  - archmage.png       (Boss - Wave 10)
  - war_elephant.png   (Boss - Wave 15)
  - dragon_rider.png   (Boss - Wave 20)

/map/
  - grass.png    (Grass tile - seamless)
  - path.png     (Cobblestone path tile - seamless)
  - tavern.png   (Merlin's Beard tavern building)
  - tree.png     (Decorative tree)
  - rock.png     (Decorative rock cluster)

/projectiles/
  - arrow.png    (Arrow projectile)
  - fireball.png (Fire projectile)
  - magic.png    (Magic bolt projectile)


NOTES:
------
- The game will use canvas-drawn fallback graphics if sprites are missing
- Isometric style recommended (as shown in the reference image)
- All sprites should face toward the camera (isometric front view)
- Use consistent lighting direction (top-left)
- Maintain similar art style across all sprites for consistency
